NPD Stage-Gate

From idea to launch,
without the chaos.

A Kanban-style pipeline from Discovery through Launch with configurable gate criteria. Gate 5 GO automatically triggers a release request — no manual handoffs, no lost emails.

  • Discovery → Feasibility → Development → Validation → Launch
  • Must-meet and should-meet gate criteria per stage
  • Formal gate review sign-offs with multi-role approval
  • Automatic release trigger on Gate 5 GO

Regulatory Labeling

Click Generate.
Label done.

Tatva recursively walks your entire formula tree, sorts ingredients by descending weight, auto-detects allergens from material attributes, and populates the complete regulatory label — fully FSSAI, EU 1169, and FDA compliant.

  • FSSAI / EU 1169 / FDA compliant ingredient declarations
  • Allergens auto-detected from material master attributes
  • Nutrition panel, shelf life, country of origin, batch format
  • Regenerate instantly when formula changes
tatva.com/labeling

Self-Hosted

Production in 3 commands.

No cloud. No SaaS fees. No vendor relationship. Your server, your data, your platform — forever.

1
Clone & Configure
Clone the repo, copy the production env file, and set your PostgreSQL password and JWT secret. That's it.
2
Launch with Docker
One command starts the full stack — Nginx, React SPA, Express API, PostgreSQL, and Redis.
3
Load Demo Data
Seed realistic demo data to explore formulas, NPD projects, changes, and labels in minutes.
Terminal 
# 1. Clone
git clone https://github.com/PraveenSampathPLM/TatvaPLM.git && cd TatvaPLM

# 2. Configure — set POSTGRES_PASSWORD and JWT_SECRET
cp .env.production.example .env.production

# 3. Launch the full stack
docker compose -f docker-compose.prod.yml --env-file .env.production up -d --build

# Load demo data (optional)
docker exec plm-backend sh -c "npm run seed:demo -w @plm/backend"

# ✓ Open http://localhost — done.
